# 💸 Bill Splitter (Python)

A clean, efficient Python application designed to take the awkward math out of group dining, shared rent, and trips. Input the total, add your friends, account for tax/tip, and split the bill fairly in seconds right from your terminal or browser.

## ✨ Features

*   **Dynamic Splitting:** Split evenly among everyone or customize shares for individuals who ordered extra (or less!).
*   **Tax & Tip Calculator:** Automatically calculates and distributes tax and tip percentages proportionally across everyone's final total.
*   **Itemized Breakdown:** Add specific items, assign them to specific people, and see exactly who owes what.
*   **Export/Share Summaries:** Generates a clean text breakdown ready to be copied into a group chat.
*   **Robust Input Handling:** Gracefully handles typos, negative numbers, and empty inputs so the app doesn't crash mid-calculation.
